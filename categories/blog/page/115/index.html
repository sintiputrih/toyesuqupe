64-Layer V-NAND, 860 MB/s Reads

published on August 12, 2024
Samsung on Tuesday announced that it had initiated volume production of UFS NAND flash memory chips with 512 GB capacity based on its latest 64-layer 512 Gb V-NAND. The new UFS packages employ Samsung’s latest controller, which sport some performance improvements over its predecessors. The new UFS devices are expected to be used for various applications, including smartphones, tablets, standalone VR headsets, TVs and other. Samsung’s 512 GB eUFS device is the company’s first embedded UFS storage solution to use the latest 64-layer 512 Gb TLC V-NAND along with the new controller that features an improved NAND mapping table management.
